# Random Forest - Wine Quality Prediction

Projet de TP en Data Science utilisant l’algorithme Random Forest pour la prédiction de la qualité du vin (régression et classification).

## Contenu
- Untitled.ipynb : Notebook avec toute l’analyse
- RandomForest.pdf : Rapport final
- requirements.txt : Bibliothèques utilisées

## Méthodes
- Prétraitement avec StandardScaler
- Random Forest (régression & classification)
- GridSearchCV (k=5)
- Sélection de variables
- Comparaison avec Gradient Boosting
- Métriques : MAE, MSE, RMSE, R², Accuracy, Precision, Recall, F1-score, ROC
